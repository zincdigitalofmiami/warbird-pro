**Step 1: Study v0's ZigZag fib engine**

Read the v0 indicator's `fibZzUpdate()` function (lines ~150-170 of v0). It uses:
```pine

```
with `zigzag.Settings`, `zigzag.newInstance()`, `fibZZ.update()`, `fibZZ.lastPivot()`.

The key difference from current v2:
- v2 uses `ta.highest(high, N)` / `ta.lowest(low, N)` — repaints as new bars form
- v0 uses ZigZag confirmed pivots — anchor only changes when a NEW structural pivot is confirmed

**Step 2: Replace the anchor engine**

Remove the current confluence-scoring anchor selection (lines ~130-230 of v2). Replace with:

1. ZigZag pivot detection with ATR-based deviation threshold
2. Confirmed pivots only (ZigZag `lastPivot()` with `start.price` and `end.price`)
3. Keep the 5-lookback confluence scoring as a QUALITY metric (not anchor selector)
4. The ZigZag anchor determines WHERE fibs are drawn
5. The confluence score determines HOW STRONG the anchor is (AG feature)

```pine
// ZigZag-based fib anchor (non-repainting)
var fibSettings = zigzag.Settings.new(fibDeviation, fibDepth, color(na), false, false, false, false, "Absolute", true)
var fibZZ = zigzag.newInstance(fibSettings)
var zigzag.Pivot lastPvt = na

fibSettings.devThreshold := ta.atr(10) / close * 100 * fibDeviation

if fibZZ.update()
    lastPvt := fibZZ.lastPivot()

// Extract confirmed anchor prices
if not na(lastPvt)
    fibAnchorHigh := math.max(lastPvt.start.price, lastPvt.end.price)
    fibAnchorLow  := math.min(lastPvt.start.price, lastPvt.end.price)
    anchorHighBar := lastPvt.start.price > lastPvt.end.price ? lastPvt.start.barIndex : lastPvt.end.barIndex
    anchorLowBar  := lastPvt.start.price > lastPvt.end.price ? lastPvt.end.barIndex : lastPvt.start.barIndex
```

**Step 3: Add ZigZag inputs**


### Task 3: Add trade state machine (SL/ENTRY/TP1/TP2)

**Files:**
- Modify: `indicators/v2-warbird-unified.pine`
- Reference: Trade Manager [AlgoScopes], Position Tracker [GG_ALGO] patterns

**Step 1: Define trade state enum**

```pine
// Trade state machine
var int TRADE_NONE = 0
var int TRADE_SETUP = 1    // Setup detected, awaiting entry
var int TRADE_ACTIVE = 2   // In trade, monitoring TP/SL
var int TRADE_HIT_TP1 = 3  // TP1 reached
var int TRADE_HIT_TP2 = 4  // TP2 reached
var int TRADE_STOPPED = 5  // SL hit
var int TRADE_EXPIRED = 6  // Setup expired without entry

var int tradeState = TRADE_NONE
var float entryPrice = na
var float slPrice = na
var float tp1Price = na
var float tp2Price = na
var int setupBar = na
var int entryBar = na
```

**Step 2: Define SL/ENTRY/TP levels from fib structure**

Per locked rules:
- SL = -0.236 fib extension (fibPrice(-0.236))
- ENTRY = decision zone (0.618 level for longs, 0.382 for shorts)
- TP1 = 1.236 fib extension
- TP2 = 1.618 fib extension
- TP3 = 2.0 fib extension

```pine
// Compute trade levels
slLevel = fibBull ? fibPrice(-0.236) : fibPrice(1.236)
entryLevel = fibBull ? pZoneHi : pZoneLo  // .618 for longs, .382 for shorts
tp1Level = fibBull ? fibPrice(1.236) : fibPrice(-0.236)
tp2Level = fibBull ? fibPrice(1.618) : fibPrice(-0.618)
tp3Level = fibBull ? fibPrice(2.0) : fibPrice(-1.0)
```

**Step 3: Implement state transitions**

```pine
if barstate.isconfirmed
    // NONE → SETUP: price enters decision zone
    if tradeState == TRADE_NONE and zoneReady
        if fibBull and close <= entryLevel and close >= zoneLower
            tradeState := TRADE_SETUP
            setupBar := bar_index
        if not fibBull and close >= entryLevel and close <= zoneUpper
            tradeState := TRADE_SETUP
            setupBar := bar_index

    // SETUP → ACTIVE: price confirms entry (close inside zone + regime aligned)
    if tradeState == TRADE_SETUP and regimeAligned
        entryPrice := close
        slPrice := slLevel
        tp1Price := tp1Level
        tp2Price := tp2Level
        tradeState := TRADE_ACTIVE
        entryBar := bar_index

    // ACTIVE → TP1/TP2/STOPPED
    if tradeState == TRADE_ACTIVE
        if fibBull
            if low <= slPrice
                tradeState := TRADE_STOPPED
            else if high >= tp2Price
                tradeState := TRADE_HIT_TP2
            else if high >= tp1Price
                tradeState := TRADE_HIT_TP1
        else
            if high >= slPrice
                tradeState := TRADE_STOPPED
            else if low <= tp2Price
                tradeState := TRADE_HIT_TP2
            else if low <= tp1Price
                tradeState := TRADE_HIT_TP1

    // SETUP expiry (no entry within N bars)
    if tradeState == TRADE_SETUP and not na(setupBar) and bar_index - setupBar > 12
        tradeState := TRADE_EXPIRED

    // Reset after outcome
    if tradeState >= TRADE_HIT_TP1
        tradeState := TRADE_NONE
        entryPrice := na
        slPrice := na
        tp1Price := na
        tp2Price := na
```

**Step 4: Draw SL/ENTRY/TP lines on chart**

Only when trade is SETUP or ACTIVE:
```pine
var line slLine = line.new(na, na, na, na, color=color(na))
```
